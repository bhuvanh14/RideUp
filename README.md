# RideUp Project

An online bike parts store built with React and Node.js.

## How to Run Locally

1. Clone the repo:  
   `git clone https://github.com/yourusername/RideUp.git`

2. Install dependencies:  
   `npm install`

3. Run the frontend:  
   `cd frontend`  
   `npm start`

4. Run the backend:  
   `cd server`  
   `node server.js`

## Features

- Browse bike parts  
- Add to cart  
- Purchase online

## Tech Stack

- React  
- Node.js  
- Express  
- MongoDB
